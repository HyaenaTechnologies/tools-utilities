# Commands and Flags

## Commands

| Command        | Description                      |
| -------------- | -------------------------------- |
| clean          | Git Garbage Collection           |
| help           | Print Commands and Flags         |
| prune          | Git Prune                        |
| version        | Print Version Number             |
| update         | Git Pull                         |

## Flags

| Flag | Description                      |
| ---- | -------------------------------- |
| --c  | Git Garbage Collection           |
| --h  | Print Commands and Flags         |
| --p  | Git Prune                        |
| --v  | Print Version Number             |
| --u  | Git Pull                         |
