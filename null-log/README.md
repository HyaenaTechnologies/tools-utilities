# Null Log

Clean Log File Contents - Retain Log Files

## Features

- Clean Logs
- Retain Logs

## Build

- [Rust][Rust Language]

```shell
git clone

cargo check

cargo build --release --target x86_64-unknown-linux-gnu

mv ./target/x86_64-unknown-linux-gnu/release/null-log ./binary

sudo ./binary/null-log
```

## Install System Update Daemon

```shell
mv ./null-log ~/
```
